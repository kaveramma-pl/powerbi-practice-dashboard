Power BI Practice Dashboard

Project Overview

This project is a multi-page Power BI dashboard created as part of guided practice using a sample dataset. It demonstrates how to build interactive reports with multiple views to analyze business performance across different dimensions.

Tools Used

Power BI Desktop

Dataset

Sample dataset used for practice and learning purposes

Dashboard Structure

The report consists of multiple pages focusing on different areas:

Overview dashboard with key KPIs and sales insights
Product-focused analysis including top and bottom performing products
Employee-level dashboard
Finance dashboard
Detailed product view

Dashboard Features

KPI cards displaying Total Orders, Total Sales, and Profit Margin
Interactive slicers for Year, City, Employee, and Product Line
Sales comparison across time
Geographical analysis using map visualization
Product-level insights using bar charts and treemaps
Navigation across multiple report pages

Analysis Highlights

Compared sales performance across years and months
Identified top and bottom selling products
Analyzed product line contribution to overall sales
Explored customer distribution across regions
Examined performance across different business perspectives

Learning Outcome

Gained experience in building multi-page dashboards in Power BI
Learned how to structure reports for different business views
Improved skills in using slicers, maps, and advanced visuals
Understood how to design user-friendly and interactive dashboards

Note

This dashboard was created by following a guided tutorial and is intended for learning purposes. It showcases understanding of Power BI features and dashboard design.
